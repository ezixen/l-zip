### L-ZIP Quick Reference
- **ACT:** Set role/persona
- **OBJ:** Primary objective
- **LIM:** Constraints/limits
- **CTX:** Background context
- **OUT:** Output format
- **SUM:** Summarize/list
- **GEN:** Generate
- **THINK:** Step-by-step reasoning
- **EVAL:** Evaluate/critique
- **VIS:** Visualization
- **=>:** Leads to/results in
- **@:** Time, level, audience
- **LEN:** Length limit
